## Dependencies

1.  [Maven Apache](https://maven.apache.org/)
2.  [Oracle JDK](http://www.oracle.com/technetwork/java/javase/downloads/index.html)

## Use

#### - Clone this repository
```sh
git clone https://github.com/binhonglee/LibrarySystem.git
```

#### - Run the program
```sh
mvn compile
mvn exec:java
```

#### - Run test
```sh
mvn test
```
